Accountability: About This Data Set

Mobile data connected by AWARE framework on Jieli's phone

Where did this data come from?

The data is collected by Jieli's Huawei Mate 9 cell phone. All data all collected using the AWARE framework, an Open-source Context Instrumentation Framework For Everyone. Collection time was from Feb 19 to Mar 6 2018 in Pittsburgh, PA.

Who has access to it?

All plots are made public on Tableau Public. I also collected 5409020 records of accelerometer, 3265684 records of magnetometer, and 2567022 records of gyroscope and many others.
